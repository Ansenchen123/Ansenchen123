# Ansen Chen - I build applied AI and practical software tools.

I build usable software with Python, C++, TypeScript, JavaScript, and applied AI workflows, usually turning a local workflow or experiment into a runnable tool. My projects cover document automation, reusable-container service flows, factory maintenance utilities, encrypted local-first apps, AI chat clients, cybersecurity education, and trading workflow simulations. I care about clear setup paths, local data handling, tests around important boundaries, and interfaces that make the core workflow easy to evaluate.

## 摘要

- 這份精選列表目前涵蓋 10 個專案：PDF2PPt、markerthon、trading-system、factor、quiet-diary、factory-manager-portable、AI_server_test、AI_local、information-security、backtesting-project-collection。
- PDF2PPt 聚焦文件自動化與可編輯簡報重建；markerthon 是團隊黑客松的循環容器服務 monorepo，串接 FastAPI 後端、商家端與政府端儀表板。
- trading-system 展示台股自動化工作流程，包含 Node.js 編排、Python broker service、本機 dashboard、風控邊界與測試；回測專案則整理多個 Python 策略驗證實驗。
- factor 與 factory-manager-portable 都服務工廠維護場景，一個偏 C++ 原生桌面工具，一個偏 Electron 可攜式管理工具，重點是本機資料、提醒與可交付流程。
- quiet-diary 是 PIN 與 AES-GCM 保護的本機加密日記 PWA，支援離線、IndexedDB、備份匯入匯出與 Capacitor Android 測試包裝。
- AI_server_test 與 AI_local 呈現雲端和本機 AI 聊天工作流；information-security 以互動網站、示範與小遊戲說明基礎資安觀念。
- 整體方向是把實驗型想法整理成容易安裝、容易理解、能被產品與技術角色共同評估的成果。
